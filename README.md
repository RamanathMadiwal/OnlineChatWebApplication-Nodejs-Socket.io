Procedure:-<br>
Step 1:Download the files.<br>
Step 2:Install npm.<br>
Step 3:Install below modules with exact version as specified in package.json.<br>
 "express": "^4.15.2"<br>
 "moment": "^2.18.1"<br>
 "socket.io": "^1.7.3"<br>
Step 4:start the server with node server/server.js<br>
